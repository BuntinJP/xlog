# About This Blog

これは私の技術ブログ的なものです。私的なものも書くかもしれないので、かなり自己満成分が強いです。

備忘録を最終的にまとめるところをいずれ作る予定でした。zenn や Qiita は他の記事がよく目にチラついてなんかやる気がなくなるので、やっぱり自分で作って一人でやる感じがいいかなと思います。

記事内容的には、技術的な日記になると思います。自分が忘れないように作業過程をメモっておくという意味で備忘録です。git はなんだかんだコミットメッセージじゃ当時の環境とか何考えてるかとかなんもわからないので、正直意味はないです。

ということで、そこらへんで参考になるものがあれば幸いです。
